## Default Permission

Default permissions for the WebDriver plugin

#### This default permission set includes the following:

- `allow-resolve`

## Permission Table

<table>
<tr>
<th>Identifier</th>
<th>Description</th>
</tr>


<tr>
<td>

`webdriver-automation:allow-resolve`

</td>
<td>

Enables the resolve command without any pre-configured scope.

</td>
</tr>

<tr>
<td>

`webdriver-automation:deny-resolve`

</td>
<td>

</td>
</tr>
</table>
